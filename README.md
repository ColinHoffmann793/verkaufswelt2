# 🎮 Verkaufswelt — Deluxe Edition

Die zweite, **grafisch komplett überarbeitete** Version des DISG-Verkaufstrainings. Gleiches Spielprinzip wie [Version 1](https://colinhoffmann793.github.io/verkaufswelt/) — erkenne den Kundentyp und antworte typgerecht — aber mit „insane" Grafik und deutlich mehr Spieltiefe.

## ▶️ Jetzt spielen
👉 **https://colinhoffmann793.github.io/verkaufswelt2/**

## ✨ Neu in der Deluxe Edition
- **Tag-/Nacht-Zyklus** (jeder Verkauf rückt die Zeit vor → Sternennacht + großes Abschluss-Feuerwerk)
- **Partikel:** Springbrunnen, Schmetterlinge, Vögel, fallendes Laub, Glühwürmchen, Feuerwerk
- **Größere Welt:** Bergpanorama, Wald, Gras, Kiosk, mehr Deko · **Kino-Kamera** im Gespräch · **Sprint** (Shift)
- **XP & Level** (Trainee → Sales Legend) · **Aufträge** · **Kiosk-Shop** (Power-ups) · **Statistik** · **Achievements** · **Minimap** · **Sound**
- **Versteckte Easter Eggs** 🥚 — augen auf!

## 🕹️ Steuerung
**Klick ins Bild** → Maus-Blick · **WASD** gehen · **Shift** rennen · **E** interagieren · **Esc** Maus frei · Handy: ziehen + „Go"-Knopf

## ⚙️ Technik
Eine einzige `index.html` (Three.js / WebGL eingebettet) — läuft komplett offline im Browser.
